# rollup-ts-template

你可以五步内对ts项目完成打包

## Get started

## 1. 创建文件夹

> mkdir FolderName

## 2. 进入文件夹

> cd FolderName

## 3. 下载模板

> git clone `git@github.com:mekefly/rollup-ts-template.git` ./

或

> git clone `https://github.com/mekefly/rollup-ts-template.git` ./

## 4. 安装依赖

> pnpm install

## 5. 打包

> pnpm build
